# Граф по списку ребер
ограничение по времени на тест: 1 секунда  
ограничение по памяти на тест: 256 мегабайт  
ввод: стандартный ввод  
вывод: стандартный вывод

Граф — это множество вершин ***V*** и множество ребер ***E***. Каждое ребро
соединяет пару вершин.

Неориентированный граф — это граф, в котором у ребер нет ориентации. Иными
словами ребро ***(a,b)*** симметрично соединяет вершину ***a*** и вершину
***b***. Ребра ***(a,b)*** и ***(b,a)*** идентичны.

Пусть ***N*** — это количество вершин графа, а ***M*** — это количество ребер.
(***N*** = |***V***|; ***M*** = |***E***|)

Матрица смежности графа — это матрица ***A*** с размерами ***N×N***, в которой
каждый элемент ***A\[x,y\]*** равен количеству ребер между вершинами ***x*** и
***y***.

Матрица инцидентности графа — это матрица ***A*** с размерами ***N×M***, в
которой каждый элемент ***A\[x,y\]*** равен **1**, если ребро ***y*** инцидентно
вершине ***x***, и равен **0** в ином случае. Ребро инцидентно вершине, если
вершина является одним из концов этого ребра.

Списки смежности графа — это ***N*** списков смежности, по одному для каждой
вершины графа. Список смежности вершины — это перечисление всех вершин, смежных
заданной.

Вам задан неориентированный граф в виде списка ребер. Вычислите для него матрицу
смежности, матрицу инцидентности и списки смежности.

## Входные данные
В первой строке задается одно целое число ***N*** (**2** ⩽ ***N*** ⩽ **32**)
— количество вершин в графе.

В первой строке задается одно целое число **M**
(**1** ⩽ ***M*** ⩽ ***N***⋅(***N***−**1**)/**2**) — количество ребер в графе.

В следующих ***M*** строках задается по **2** целых числа ***ai***, ***bi***
(**1** ⩽ ***ai***, ***bi*** ⩽ ***N***; ***ai≠bi***) — вершины, которые соединяет
***i***-е ребро.

Гарантируется, что ни одна пара вершин не соединена более чем одним ребром.

## Выходные данные
В первой строке выведите *ADJACENCY MATRIX*

В следующих ***N*** строках выведите по ***N*** целых чисел ***A\[i,j\]*** через
пробел — матрицу смежности заданного графа.

В следующей строке выведите *INCIDENCE MATRIX*

В следующих ***N*** строках выведите по ***M*** целых чисел ***I\[i,k\]*** через
пробел — матрицу инцидентности заданного графа. Ребра должны быть пронумерованы
(и перечисляться в матрице) в порядке следования во входных данных.

В следующей строке выведите *ADJACENCY LISTS*

В каждой из следующих ***N*** строках выведите список смежности для очередной
вершины графа в формате

*V => A1 A2 A3 ...*

где *V* — это номер очередной вершины, а *A1 A2 A3 ...* — это перечисление через
пробел вершин, смежных с *V*. Смежные вершины должны перечисляться в порядке
возрастания.
