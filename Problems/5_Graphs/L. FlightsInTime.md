# Рейсы во времени
ограничение по времени на тест: 2 секунды  
ограничение по памяти на тест: 256 мегабайт  
ввод: стандартный ввод  
вывод: стандартный вывод

Между $N$ населенными пунктами совершаются пассажирские рейсы на машинах
времени.

В момент времени $0$ вы находитесь в пункте $A$. Вам дано расписание рейсов.
Требуется оказаться в пункте $B$ как можно раньше (то есть в наименьший
возможныймомент времени).

При этом разрешается делать пересадки с одного рейса на другой. Если вы
прибываете в некоторый пункт в момент времени $T$, то вы можете уехать из него
любым рейсом, который отправляется из этого пункта в момент времени $T$ или
позднее (но не раньше).

## Входные данные
Первая строка содержит число $N$ $(1 \leq N \leq 1000)$ — количество населенных
пунктов. Вторая строка содержит два целых числа $A,B$ $(1 \leq A,B \leq N)$
— номера начального и конечного пунктов. Третья строка содержит число $K$
$(0 \leq K \leq 1000)$ — количество рейсов. Следующие $K$ строк содержат
описание рейсов, по одному в каждой строке. Каждое описание представляет собой
четыре целых числа. Первое число описания задает задает номера пункта
отправления, второе — время отправления, третье — пункт назначения, четвертое
— время прибытия. Номера пунктов — натуральные числа из диапазона от $1$ до $N$.
Пункт назначения и пункт отправления могут совпадать. Время измеряется в
некоторых абсолютных единицах и задается целым числом, по модулю не
превосходящим $10^9$. Поскольку рейсы совершаются на машинах времени, то время
прибытия может быть как больше времени отправления, так и меньше или равным ему.

Гарантируется, что входные данные таковы, что добраться из пункта $A$ в пункт
$B$ всегда возможно.

## Выходные данные
Выведите одно целое число — минимальное время, когда вы сможете оказаться в
пункте $B$.
