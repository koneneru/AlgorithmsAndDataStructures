# Шириивук
ограничение по времени на тест: 1 секунда  
ограничение по памяти на тест: 256 мегабайт  
ввод: стандартный ввод  
вывод: стандартный вывод

Исследователей языков Галактики интересует диковинный язык вуки – расы волосатых
двуногих гуманоидов, которые живут на лесной планете Кашиик и имеют особое
строение голосовых связок. Для чужаков этот язык обычно звучит как звериные рёв,
рычание, мычание и стоны, не выражающие ничего, кроме эмоций. Предположительно,
он специально сделан таким для усложнения понимания представителями других рас.
Однако, шириивук на самом деле столь же информативен, как и основной
галактический язык. В шириивуке существует более 150 слов для обозначения
дерева в зависимости от его типа, строения, сухости и внутренней структуры.

Исследователи хотят выделить составные части слов языка, используя доступные им
данные. Вашей задачей сейчас является поиск такого набора частей слов – слогов,
что каждое известное слово на шириивуке может быть разбито на слоги из этого
набора (один и тот же слог при этом можно использовать несколько раз).

Конечно, таких наборов может быть очень много. Среди всех таких наборов нужно
найти такой, что суммарная длина всех слогов минимальна. Если таких наборов
несколько, то среди них нужно найти такой, что количество слогов в нем
минимально. Можно показать, что существует только один такой набор.

## Входные данные
В первой строке задается одно целое число $N$ $(1 \leq N \leq 50)$ – количество
известных слов на шириивук.

В последующих $N$ строках задается по одному слову из языка. Каждое слово – это
строка, состоящая из строчных букв латинского алфавита, длина которой не
превосходит **50** символов. Все слова различны.

## Выходные данные
Выведите одно целое число $M$ – количество слогов в найденном наборе. Далее
выведите $M$ слогов в алфавитном (лексикографическом) порядке. Каждый слог
должен быть строкой, состоящей из строчных букв латинского алфавита. Вы можете
разделять части ответа пробелами или переводами строк.

## Пример
**Входные данные**
```no-hilights
4
wyoaoaoa
wyng
rumprum
wyngwyng
```
**Выходные данные**
```no-hilights
5
ng
oa
p
rum
wy
```

## Примечание
В первом примере набор состоит из **5** слогов с суммарной длиной **10**:

Каждое из слов можно разбить на слоги из этого набора:
 1. wyoaoaoa = wy + oa + oa + oa
 1. wyng = wy + ng
 1. rumprum = rum + p + rum
 1. wyngwyng = wy + ng + wy + ng
