# 2-3 лестница
ограничение по времени на тест: 1 секунда  
ограничение по памяти на тест: 256 мегабайт  
ввод: стандартный ввод  
вывод: стандартный вывод

Вася очень торопится домой, поэтому поднимаясь по лестнице из ***N*** ступенек
он каждый раз перепрыгивает через одну или две ступеньки (то есть сначала он
прыгает на **2**-ю или на **3**-ю ступеньку, со **2**-й он может прыгнуть на
**4**-ю или на **5**-ю и т.д.) Требуется определить количество способов,
которыми он может попасть на последнюю, ***N***-ю ступеньку.

## Входные данные
Вводится единственное целое число ***N*** (**1** ⩽ ***N*** ⩽ **150**).

## Выходные данные
Выведите единственное число — искомое количество способов.

## Примеры
**Входные данные**
```no-hilights
4
```
**Выходные данные**
```no-hilights
1
```
**Входные данные**
```no-hilights
5
```
**Выходные данные**
```no-hilights
2
```
**Входные данные**
```no-hilights
150
```
**Выходные данные**
```no-hilights
856798505175074247
```
