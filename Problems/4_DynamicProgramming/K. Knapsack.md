# Рюкзак
ограничение по времени на тест: 1 секунда  
ограничение по памяти на тест: 256 мегабайт  
ввод: стандартный ввод  
вывод: стандартный вывод

У вас есть ***n*** золотых слитков с весами ***w1***, ***w2***, ..., ***wn***.
Также у вас есть рюкзак, в который можно поместить любое количество золотых
слитков с суммарным весом не более ***S***. Определите максимальный суммарный
вес золотых слитков, которое можно унести в рюкзаке.

## Входные данные
В первой строке заданы два целых числа ***S*** и ***n***
(**1** ⩽ ***S*** ⩽ **10^4**; **1** ⩽ ***n*** ⩽ **300**) — максимальный
переносимый вес в рюкзаке и количество золотых слитков.

Во второй строке заданы ***n*** целых чисел ***w1***, ***w2***, ..., ***wn***
(**1** ⩽ ***wi*** ⩽ **10^5**) — веса золотых слитков.

Выходные данные
Выведите одно целое число — максимальный суммарный вес золотых слитков, которое
можно унести в рюкзаке.

## Примеры
**Входные данные**
```no-hilights
10 3
5 7 4
```
**Выходные данные**
```no-hilights
9
```
**Входные данные**
```no-hilights
10 3
11 13 17
```
**Выходные данные**
```no-hilights
0
```

## Примечание
В первом примере можно взять слитки с весами **4** и **5**, но оставить слиток
весом **7**. Добиться суммарного веса ровно **10** нельзя.

Во втором примере ни один из слитков нельзя унести в рюкзаке.
