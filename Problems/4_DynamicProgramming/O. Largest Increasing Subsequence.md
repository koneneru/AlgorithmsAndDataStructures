# Наибольшая возрастающая подпоследовательность
ограничение по времени на тест: 1 секунда  
ограничение по памяти на тест: 256 мегабайт  
ввод: стандартный ввод  
вывод: стандартный вывод

Задана последовательность из ***N*** целых чисел ***X1***, ***X2***, ...,
***XN***. Требуется вычеркнуть из них минимальное количество чисел так, чтобы
оставшиеся шли в порядке строгого возрастания.

## Входные данные
В первой строке задается одно целое число ***N*** (**1** ⩽ ***N*** ⩽ **2⋅10^5**)
— количество чисел.

Во второй строке через пробел задается ***N*** целых чисел ***X1***, ***X2***,
..., ***XN*** (**1** ⩽ ***Xi*** ⩽ **10^9**) — числа последовательности.

## Выходные данные
Выведите одно целое число — количество невычеркнутых чисел, формирующих
наибольшую возрастающую подпоследовательность.

## Примеры
**Входные данные**
```no-hilights
6
2 5 3 4 6 1
```
**Выходные данные**
```no-hilights
4
```
**Входные данные**
```no-hilights
9
1 1 1 2 2 2 3 3 3
```
**Выходные данные**
```no-hilights
3
```
