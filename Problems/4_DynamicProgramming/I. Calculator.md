# Калькулятор
ограничение по времени на тест: 1 секунда  
ограничение по памяти на тест: 256 мегабайт  
ввод: стандартный ввод  
вывод: стандартный вывод

Имеется калькулятор, который может выполнять следующие действия:
- умножить число ***X*** на **2**;
- умножить число ***X*** на **3**;
- увеличить число ***X*** на **1**;

Определите, какое наименьшее количество операций потребуется для того, чтобы
получить из числа **1** число ***N***.

## Входные данные
В единственной строке задается одно целое число ***N***
(**2** ⩽ ***N*** ⩽ **10^6**) — требуемое число.

## Выходные данные
В первой строке выведите одно целое число ***k*** — минимальное количество
операций.

Во второй строке через пробел выведите ***k+1*** целых чисел ***a1***, ***a2***,
..., ***ak+1*** — числа, последовательно получающиеся при выполнении операций.
Первое число должно быть равно **1**, последнее число должно быть равно ***N***.
Каждое число начиная со второго должно быть получено из предыдущего с помощью
одной из операций калькулятора.

## Примеры
**Входные данные**
```no-hilights
1
```
**Выходные данные**
```no-hilights
0
1
```
**Входные данные**
```no-hilights
5
```
**Выходные данные**
```no-hilights
3
1 2 4 5
```
**Входные данные**
```no-hilights
962340
```
**Выходные данные**
```no-hilights
17
1 2 6 18 54 55 110 220 660 1980 5940 17820 17821 53463 160389 160390 320780 962340
```
