# Максимальный подпалиндром
ограничение по времени на тест: 1 секунда  
ограничение по памяти на тест: 256 мегабайт  
ввод: стандартный ввод  
вывод: стандартный вывод

Вам задана строка ***S*** состоящая из N заглавных букв латинского алфавита.

Найдите наибольшую длину подстроки ***S***, которая является палиндромом.

Подстрока ***S***\[***L***;***R***\] строки ***S*** — это строка, состоящая из
всех символов строки ***S*** начиная с позиции ***L*** и заканчивая позицией
***R*** в том же порядке. Иными словами подстрока — это какой-то
последовательный отрезок оригинальной строки.

Палиндром — то строка, которая одинаково читается как справа-налево, так и
слева-направо. Палиндром — это строка, которая не изменится, если ее перевернуть.

## Входные данные
В единственной строке задается строка ***S***, содержащая от **1** до **2000**
заглавных букв латинского алфавита.

## Выходные данные
Выведите одно целое число — наибольшую длину подстроки ***S***, которая является
палиндромом.

## Примеры
**Входные данные**
```no-hilights
ABC
```
**Выходные данные**
```no-hilights
1
```
**Входные данные**
```no-hilights
ABACABADABAC
```
**Выходные данные**
```no-hilights
9
```
