## Три единицы подряд
ограничение по времени на тест: 1 секунда  
ограничение по памяти на тест: 256 мегабайт  
ввод: стандартный ввод  
вывод: стандартный вывод

По заданному числу ***N*** определите количество последовательностей из нулей
и единиц длины ***N***, в которых никакие три единицы не стоят подряд.

## Входные данные
В единственной строке задается одно целое число ***N*** (**1** ⩽ ***N*** ⩽ **30**)
— требуемая длина последовательностей.

## Выходные данные
Выведите одно целое число — количество искомых последовательностей.

## Примеры
**Входные данные**
```no-hilights
4
```
**Выходные данные**
```no-hilights
13
```
**Входные данные**
```no-hilights
5
```
**Выходные данные**
```no-hilights
24
```
**Входные данные**
```no-hilights
24
```
**Выходные данные**
```no-hilights
2555757
```

## Примечание
Рассмотрим первый пример. Всего существует **16** последовательностей длины **4**.
Среди них подходят:
1. *0000*
2. *0001*
3. *0010*
4. *0011*
5. *0100*
6. *0101*
7. *0110*
8. *1000*
9. *1001*
10. *1010*
11. *1011*
12. *1100*
13. *1101*

и не подходят:
1. *0111*
2. *1110*
3. *1111*
