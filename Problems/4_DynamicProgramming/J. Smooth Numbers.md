# Плавные числа
ограничение по времени на тест: 1 секунда  
ограничение по памяти на тест: 256 мегабайт  
ввод: стандартный ввод  
вывод: стандартный вывод

Назовём целое положительное число плавным, если разность любых двух его соседних
цифр не превосходит по модулю единицы. Вам необходимо определить количество
***N***-значных плавных чисел.

## Входные данные
Вводится единственное целое число ***N*** (**1** ⩽ ***N*** ⩽ **35**).

## Выходные данные
Выведите единственное число — искомое количество плавных чисел.

## Пример
**Входные данные**
```no-hilights
2
```
**Выходные данные**
```no-hilights
26
```
