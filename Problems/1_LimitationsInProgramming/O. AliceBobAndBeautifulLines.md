# Алиса, Боб и красивые строчки
ограничение по времени на тест: 1 секунда  
ограничение по памяти на тест: 256 мегабайт  
ввод: стандартный ввод  
вывод: стандартный вывод

Алиса и Боб хотят составить строку, состоящую из букв A и B.

Однако, если в строке есть 3 подряд идущих буквы B, то строка не нравится Алисе.
Боб менее критичен чем Алиса. Если в строке есть 4 подряд идущих буквы A, то
строка не нравится Бобу.

Алиса и Боб составили строку длины n, которая нравится им обоим. Сколько
различных строк они могли составить?

Так как ответ может быть очень большим, выведите остаток от его деления на
число 10^9+7.

## Входные данные
Одно целое число n (1 ⩽ n ⩽ 1000) — длина строки, составленной Алисой и Бобом.

## Выходные данные
Выведите количество способов составить строку из символов A и B длины n, в
которой нет трех подряд идущих символов B и четырех подряд идущих символов A по
модулю 10^9+7 (остаток от деления этого числа на 10^9+7).

## Примеры
**Входные данные**
```no-hilights
3
```
**Выходные данные**
```no-hilights
7
```
**Входные данные**
```no-hilights
4
```
**Выходные данные**
```no-hilights
12
```
**Входные данные**
```no-hilights
24
```
**Выходные данные**
```no-hilights
732777
```

## Примечание
В первом примере, среди 8 возможных строк, есть строка BBB, которая не нравится
Алисе.

Во втором примере всего существует 16 различных строк. Алисе не нравятся три
строки: ABBB, BBBA и BBBB. Бобу не нравится одна строка AAAA.
