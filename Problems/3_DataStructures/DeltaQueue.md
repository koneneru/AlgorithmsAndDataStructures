# Дельта-очередь

ограничение по времени на тест: 1.5 секунд  
ограничение по памяти на тест: 256 мегабайт  
ввод стандартный: ввод  
вывод стандартный: вывод

В этой задаче вам необходимо реализовать структуру данных типа очередь, которая  
поддерживает следующие операции:
1. добавить элемент в очередь;
2. удалить самый старый элемент из очереди (тот, который был добавлен раньше  
всех остальных);
3. вычислить разницу между самым большим и самым маленьким элементом в очереди.

## Входные данные:

В первой строке задается одно целое число q (1⩽q⩽2⋅10^5) — количество операций.

В следующей строке задается q целых чисел x1, x2,..., xn (−1⩽xi⩽10^9) — описание  
операций. Если xi = −1, то i-я операция — это удаление самого старого элемента из  
очереди, в ином случае i-я операция — это добавление в очередь  элемента со значением xi.

Гарантируется, что при выполнении операций удаления в очереди будет хотя бы один элемент.

## Выходные данные:

После каждой операции выведите одно целое число — разницу между максимальным и  
минимальным элементами в очереди в текущий момент.

Если очередь пуста, то выведите −1 вместо этого. Вы можете разделять числа  
пробелами или переводами строк.

## Пример
**Входные данные:**
```no-hilights
18
11 33 22 -1 -1 -1 20 0 32 20 90 -1 20 20 -1 -1 -1 -1
```
**Выходные данные:**
```no-hilights
 0 22 22 11 0 -1 0 20 32 32 90 90 90 90 70 70 70 0
```
