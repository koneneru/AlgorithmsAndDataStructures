# Цены в Шестерочке \[2\]
ограничение по времени на тест: 1.5 секунд  
ограничение по памяти на тест: 256 мегабайт  
ввод: стандартный ввод  
вывод: стандартный вывод  

В популярной торговой сети Шестерочка регулярно и автоматически меняются цены.
Персонал магазинов далеко не всегда успевает добавлять новые ценники и убирать
старые. Из-за этого часто возникает ситуация, когда в разных частях магазина
есть ценники с разными ценами на один и тот же товар. Это конечно же вызывает
большие проблемы на кассе.

Впрочем, как подкованный покупатель вы знаете, что вы можете требовать продать
вам товар по цене, указанной на ценнике, а если таких несколько несколько — по
любой из указанных цен. Конечно, вы хотели бы купить товары по минимальной из
указанных цен.

Ходить по всему магазину в поисках всех ценников весьма утомительно, поэтому вы
хотите автоматизировать этот процесс. Вам задается информация о всех ценниках в
магазине. Напишите программу, которая вычислит минимальную цену для всех
доступных продуктов.

## Входные данные
В первой строке задается одно целое число $n$ $(1 \leq n \leq 2⋅10^5)$
— количество ценников в магазине.

В следующих n строках задаются описания ценников. В каждой из этих строк
задается название продукта и его цена через пробел. Название продукта — это
строка, состоящая из строчных и заглавных букв латинского алфавита длиной от
$1$ до $20$ символов. Цена продукта — это одно целое число в диапазоне от $1$
до $999$.

## Выходные данные
В первой строке выведите одно целое число $x$ — количество различных товаров,
для которых есть хотябы один ценник в магазине.

В следующих $x$ строках выведите описания различных товаров. В каждой из этих
строк выведите название продукта и минимальную цену на него.

Вы можете выводить товары в любом порядке.
