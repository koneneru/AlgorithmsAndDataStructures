# Сумма на отрезке с изменениями

ограничение по времени на тест: 1.5 секунд  
ограничение по памяти на тест: 256 мегабайт  
ввод: стандартный ввод  
вывод: стандартный вывод

Вам задан массив целых чисел a1, a2, ..., an. Вам нужно последовательно
обрабатывать два вида запросов:
1. вычислить сумму всех элементов на отрезке [l; r] (включая обе границы);
2. изменить элемент на позиции p на значение v.

## Входные данные
В первой строке задается одно целое число n (1⩽n⩽2⋅10^5) — размер массива.

Во второй строке через пробел задается n целых чисел a1, a2, ..., an 
(−10^4⩽ai⩽10^4) — элементы массива a.

В третьей строке задается одно целое число q (1⩽q⩽3⋅10^5) — количество запросов.

В следующих q строках описываются запросы. По одному запросу в строке. Каждый 
запрос может быть запросом суммы или запросом изменения.

Запросы суммы имеют формат *S l r*, где *l* — это левая граница диапазона, 
а *r* — это правая граница диапазона. (1⩽ l ⩽ r ⩽n)

Запросы изменения имеют формат *A p v*, где *p* — это позиция элемента, 
а *v* — это новое значение элемента. (1⩽ p ⩽n; −10^4⩽ v ⩽10^4)

## Выходные данные
Для каждого запроса суммы на отрезке выведите одно число — сумму элементов.

Вы можете разделять числа пробелами или переводами строк.

## Пример
**Входные данные**
```no-hilight
5
-2 -1 0 1 2
12
S 1 3
S 4 5
S 1 5
A 2 7
A 3 8
A 4 9
S 1 1
S 2 2
S 3 3
S 4 4
S 5 5
S 1 5
```
**Выходные данные**
```no-hilight
-3
3
0
-2
7
8
9
2
24
```
## Примечание
В этой задаче большой размер ввода/вывода. Вам нужно использовать быстрые способы 
ввода/вывода, в ином случае ваше решение может получить Time Limit Exceeded.
