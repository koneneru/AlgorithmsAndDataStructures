# Массовое обновление \[2\]
ограничение по времени на тест: 1.5 секунд  
ограничение по памяти на тест: 256 мегабайт  
ввод: стандартный ввод  
вывод: стандартный вывод

Задан массив из n чисел, изначально заполненный нулями. Вам необходимо выполнять
c ним два типа запросов: присваивать всем элементам отрезка массива заданное
значение и отвечать, какое значение записано в соответствующей ячейке массива.

## Входные данные
В первой строке через пробел задана пара целых чисел n и q — количество ячеек
массива и количество запросов (1⩽n,q⩽2⋅10^5).

Каждая из последующих q строк содержит описание очередного запроса в одном из
форматов:
 - *= l r x* — записать во все ячейки с номерами от *l* до *r* значение *x* 
 (1 ⩽ l ⩽ r ⩽ n; 0 ⩽ x ⩽ 10^9);
 - *? i* — вывести значение элемента в ячейке номер *i* (1 ⩽ i ⩽ n ).

## Выходные данные
Для каждого запроса в формате *? i* выведите ответ на него.

Вы можете разделять числа пробелами или переводами строк.

## Пример
**Входные данные**
```no-hilight
5 7
= 2 3 10
? 3
? 4
= 3 4 20
? 2
? 3
? 4
```
**Выходные данные**
```no-hilight
10
0
10
20
20
```
