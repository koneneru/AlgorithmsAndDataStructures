# k-я порядковая статистика
ограничение по времени на тест: 2 секунды  
ограничение по памяти на тест: 256 мегабайт  
ввод: стандартный ввод  
вывод: стандартный вывод

Задано ***n*** целых чисел ***a1, a2, ..., an***. Назовём ***k***-й порядковой
статистикой среди них такое число ***x = aj*** (для некоторого j), что среди
***a1, ..., an*** существует ровно ***k−1*** число, меньше ***x***.

Напишите программу, которая по заданному набору чисел находит ***k***-ю
порядковую статистику в нём.

## Входные данные
Первая строка ввода содержит три целых числа ***n***, ***k*** и ***m***
(**1** ⩽ ***n*** ⩽ **30_000_000**; **1** ⩽ ***k*** ⩽ ***n***;
***min(n,2)*** ⩽ ***m*** ⩽ ***min(n,1000)***).

Вторая строка содержит три целых числа ***a***, ***b*** и ***c***
(**−2^31** ⩽ ***a,b,c*** ⩽ **2^31−1**).

Третья строка содержит m целых чисел: ***a1***, ***a2***, ..., ***am***
(**−2^31** ⩽ ***ai*** ⩽ **2^31−1**).

Остальные элементы могут быть вычислены по формуле:
***f(a ⋅ a(i−2) + b ⋅ a(i−1) + c)***. Здесь ***f(y)*** возвращает такое число
***z***, где **−2^31** ⩽ ***z*** ⩽ **2^31−1**, что ***y − z*** делится без
остатка на **2^32**.

## Выходные данные
Выведите ***k***-ю порядковую статистику среди ***a1***, ***a2***, ..., ***an***.

## Примеры
**Входные данные**
```no-hilights
3 2 3
0 0 0
1 2 3
```
***Выходные данные***
```no-hilights
2
```
**Входные данные**
```no-hilights
1000 10 3
1 1 1
1 2 3
```
***Выходные данные***
```no-hilights
-2081387834
```
